# 📊 Service Level Dashboard | CPFR

Dashboard desenvolvido no Power BI para monitoramento de nível de serviço logístico em operações de supply chain.

## 🎯 Objetivo
Analisar a performance de fornecedores e identificar gargalos através de indicadores como OTIF, On Time e In Full.

## 📌 Principais Métricas
- OTIF (On Time In Full)
- On Time Delivery
- In Full
- Lead Time
- Atraso Médio

## 📊 Análises Desenvolvidas
- Evolução do OTIF ao longo do tempo
- Performance por fornecedor
- Scatter plot (Lead Time vs OTIF)
- Identificação automática de fornecedor crítico

## 🧠 Insights
O dashboard permite identificar fornecedores com baixo nível de serviço e alto lead time, indicando risco operacional e impacto na cadeia de suprimentos.

## 🚀 Destaque
Inclui insight automático via DAX para identificação de fornecedor crítico.

## 🛠️ Ferramentas
- Power BI
- DAX
- Modelagem de dados
